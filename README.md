# Stylish-Glass-Morphism-Login-Form

This glassmorphism login form features:

1. A colourful animated gradient background with flowing colors (pink, purple, teal, and orange)

2. Decorative floating circle elements that enhance the design

3. A frosted glass container with blur effects and subtle transparency

4. Animated border highlight effects using pseudo-elements

5. Smooth hover animations on interactive elements

6. Modern form elements with custom styling:
   - Username/email input
   - Password input
   - Remember me checkbox
   - Forgot password link
   - Login button with hover effects
   - Registration link

The design is fully responsive and uses modern CSS techniques like backdrop-filter for the glass effect. The contrast between the text and background maintains good readability while achieving the stylish glassmorphism effect.
